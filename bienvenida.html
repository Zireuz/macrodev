📦 Instalar extensión VS Code
1
Descargar el instalador

Busca el archivo macrodev-terminal-1.0.0.vsix que viene con MacroDev.

2
Instalar en VS Code

Abre una terminal y ejecuta:

code --install-extension macrodev-terminal-1.0.0.vsix

O ve a Ctrl+Shift+X → ··· → Install from VSIX

3
Reiniciar VS Code completamente

Cierra con la X y vuelve a abrirlo. No uses "Reload Window".

4
Verificar que está activa

Deberías ver $(terminal) MacroDev :4001 en la barra inferior de VS Code.
